# Logistics SQL Data Analysis Project

## Overview

This project explores a logistics database to analyze operational performance, revenue, and efficiency using SQL.

The goal is to simulate real-world business analysis by answering key questions related to:

* Revenue generation
* Driver performance
* Fleet utilization
* Operational efficiency

---

## 🗂️ Database Schema

The dataset includes the following key tables:

* Drivers
* Trucks
* Trailers
* Customers
* Loads
* Trips
* Fuel Purchases
* Maintenance Records
* Delivery Events
* Safety Incidents

---

## 📊 Key Business Questions

### Revenue Analysis

* Which customer generates the most revenue?
* Which routes are the most profitable?

### Driver Performance

* How many trips does each driver complete?
* Which drivers generate the most revenue?

### Fleet & Cost Analysis

* What is the average truck age?
* Which trucks have the highest fuel consumption?
* What are the maintenance costs per truck?

### Operational KPIs

* What is the on-time delivery rate?
* What is the average trip duration?
